#### To execute the Python code for part 2 here are the instructions to follow:

1. Unzip the BBC dataset.
2. Install python from python.org.
3. Install Jupyter Notebook using the command "pip install notebook" in the terminal.
4. Download the "Part 2.ipynb" to your local machine.
5. Open the terminal and navigate to the directory where the "Part 2.ipynb" files are saved.
6. Launch and access the jupyter notebook.
7. Run the cells and check the output.
